# SQL Sales Analysis

Este projeto apresenta consultas SQL para análise de dados de vendas, com foco em responder perguntas de negócio sobre faturamento, categorias, regiões, produtos, meses e segmentos de clientes.

## Objetivo

Praticar SQL aplicado à análise de dados, transformando perguntas de negócio em consultas.

## Perguntas respondidas

- Qual foi o faturamento total?
- Qual categoria teve maior faturamento?
- Qual região vendeu mais?
- Quais foram os produtos com maior faturamento?
- Como as vendas variaram ao longo dos meses?
- Qual segmento de cliente gerou mais receita?
